# Student Grade Calculator with CI/CD

A simple web application built with HTML, CSS, and JavaScript to calculate a student's average grade and determine whether the student passes or fails.

This project was created to practice CI/CD using GitHub Actions and GitHub Pages.

---

## Features

- Enter 3 grades
- Calculate the average
- Display Pass or Fail
- Automatic code testing
- Automatic deployment

---

## Technologies

- HTML
- CSS
- JavaScript
- GitHub Actions
- GitHub Pages

## CI/CD Pipeline

```text
git push

↓

GitHub Actions

↓

HTML Check

↓

Deploy

↓

GitHub Pages
```

---

## How it Works

1. Write code

2. Push changes to GitHub

3. GitHub Actions automatically starts

4. The HTML code is checked

5. If the test passes, the website is deployed automatically
